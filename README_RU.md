Морской бой (консольная версия)
Классическая игра "Морской бой" для игры против компьютера в консоли.

Версия: 1.0
Автор: Viktor Pichugin (Виктор Пичугин)
Контакты: hsdpa08@gmail.com | vicvlp@mail.ru

Описание
Игра реализована на Python и выполняется в терминале.

Особенности:
Игровое поле 6×6.
Флот: 1 корабль длиной 3, 2 корабля по 2, 4 корабля по 1.
Автоматическая расстановка кораблей.
Отображение поля с помощью Unicode-символов.
Обработка ошибок и пользовательского ввода через собственные исключения.

Требования
Python 3.7+
Терминал, поддерживающий Unicode (для корректного отображения символов).

Установка и запуск
Склонируйте репозиторий:
git clone https://github.com/vicvlp/SeaBattle_ConsoleGame/

Перейдите в каталог проекта:
cd SeaBattle_ConsoleGame/

Запустите игру:
python main_ru.py

Если у вас несколько версий Python:
python3 main_ru.py

Правила игры и управление
Формат ввода координат: x y
где x — номер столбца (1..6), y — номер строки (1..6).
Для выхода из игры введите stop.
Ходы выполняются поочередно: пользователь — компьютер. При попадании ход повторяется.

Символы на поле:
▢ — пустая клетка
■ — часть корабля (на доске противника скрыто)
× — попадание по кораблю
● — промах
· — контур вокруг уничтоженного корабля

Структура проекта
main_ru.py — основной файл игры (содержит все классы и логику)
Dot — класс точки (координаты)
Ship — класс корабля (длина, ориентация, точки)
Board — игровая доска (размещение кораблей, обработка выстрелов)
Player, User, AI — классы игроков
Game — управление игровым процессом и основной цикл
Описание ключевых компонентов
Исключения:

BoardException — базовый класс исключений.
BoardOutException — выстрел за пределы доски.
BoardUsedException — выстрел в уже использованную клетку.
BoardWrongShipException — некорректное размещение корабля.

Класс Dot:
Хранит координаты x, y.
Реализует __eq__ и __repr__.

Класс Ship:
Инициализация: Ship(bow: Dot, length: int, orientation: int) (0 — горизонтальная, 1 — вертикальная).
Свойство dots возвращает список точек корабля.
Метод shooten(shot: Dot) проверяет попадание.

Класс Board:
Размер поля 6×6.
Методы: add_ship, out, shot, contour.
Отображение через __str__. Если hid=True, корабли скрываются.
Классы игроков:

Player — базовый класс; метод move.
AI — случайный выбор точки.
User — считывает координаты из ввода.

Класс Game:
Генерация случайных досок (random_board, try_generate_board).
Управление циклом игры (loop), приветствие (greet), вывод досок (draw_boards).
